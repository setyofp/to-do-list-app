To Do List

Command

  • "new" - add task into to do list.
  • "list" - list of all task.
  • "delete" - remove task from to do list.
  • "quit" - quit the app.

    • How to use:

    Open your inspect element window (or simply press F12), then switch to console tab.
    Start using to do list app by using command above.

    Note:if you can not see output at the console, please quit the app by enter "quit" command and refresh the page.
